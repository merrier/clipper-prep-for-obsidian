# Chrome Web Store Listing Draft

## English Summary

Prepare pages for cleaner Obsidian Web Clipper captures across WeChat, ByteTech, Feishu/Lark, and rendered links.

## Description

Obsidian Clipper Extended improves pages before you capture them with Obsidian Web Clipper. It does not replace the official clipper. Instead, it prepares tricky pages so the final Markdown is cleaner, more complete, and easier to use in your vault.

Use it when web pages render useful content in ways that normal clippers often miss:

- WeChat Official Account articles: normalizes lazy-loaded article images so they are easier to capture.
- ByteTech articles: mirrors embedded Lark document content into semantic article HTML for cleaner clipping.
- Feishu/Lark documents: converts rendered document blocks into a readable article mirror.
- Global Markdown links: preserves rendered links such as Lark/Feishu `data-href` links so they can clip as `[text](url)` instead of plain text.

Why install it:

- Better Markdown output from pages with complex rendered DOM.
- Fewer missing images and broken article captures.
- Cleaner links when clipping Lark/Feishu content.
- Site-specific enhancements can be turned on or off from the options page.
- Global processing is enabled by default and can also be disabled if you prefer.

Best workflow:

1. Install Obsidian Clipper Extended.
2. Enable the sites you care about in Options.
3. Open the target page.
4. Use the official Obsidian Web Clipper as usual.

This extension is intentionally independent from the official Obsidian Web Clipper project.
